# Roadmap

Nexora is built in phases, each one expanding the network's capabilities and value. Here's where we are and where we're going.

---

## Phase 1 — Foundation ✅ Completed

The core infrastructure is live. Users can register, run nodes, earn points, and refer others.

- CLI Node (Python-based, cross-platform)
- FastAPI backend with SQLAlchemy
- Uptime-based reward system (1 point/minute)
- Referral system with invite-only registration
- Basic anti-cheat (node limit, heartbeat spam, uptime validation)
- Cross-platform support: Linux, Windows, VPS, Android (Termux)

---

## Phase 2 — Network Expansion 🔄 In Progress

Stabilizing the network and opening it to a wider audience.

- Stable VPS deployment with production-grade backend
- Task system improvements (structured task types, result validation)
- Dashboard launch (web UI for node status, points, referrals)
- Public beta — broader access with monitored onboarding
- Improved logging and monitoring for backend health

---

## Phase 3 — Scaling & Value Layer 🔜 Planned

Making the network more valuable and harder to game.

- Real task marketplace — nodes can pick up and complete meaningful tasks
- Advanced anti-cheat — behavioral analysis, pattern detection, multi-account correlation
- Performance-based rewards — bonus multipliers for consistent high-uptime nodes
- Token integration (optional) — points-to-token conversion mechanism
- Leaderboards and node reputation scores

---

## Phase 4 — Ecosystem 🔮 Future

Opening Nexora to third-party builders and moving toward decentralization.

- Developer API — allow third parties to submit tasks to the network
- Third-party integrations — connect Nexora nodes to external platforms and services
- Decentralized expansion — explore on-chain components for reward distribution and governance
- Community governance — node operators participate in network decisions

---

> **Note:** Roadmap timelines are intentionally not date-bound. Phases are completed when they are stable and ready, not on a fixed schedule. Follow the GitHub repository for the latest updates.
